To install the theme...

  1. Upload the theme folder to the themes directory on your server. Make sure you upload the correct folder. *NOTE : The correct folder is the folder that contains the "theme.html" file, look inside the folder you are uploading and make sure its the correct folder, an extra folder may be created in the unzipping process, so CHECK!!
  2. Log into your site as administrator and navigate to the administration menu >> system >> templates. Use the template upload feature to upload the tar.gz file included in the themes zip file.
  3. Go to administration >> system >> preferences>> general and set both the theme and templates as default.

If you wish to make modifications to the theme after you have uploaded it:

Navigate to administration >> system >> preferences>> general and check "yes" where it says "Update module template .html files from themes/your theme/templates directory?" .. your updates will not show otherwise. (you should set this back to "no" when you have finished making modifications to the theme)

More...

You can set the width of the theme to your liking. By default it is set to 80%... You may want to set it to a pixel value so that you have control over the overall look of your site.

To change the width of the theme to a pixel value:
Open the style.css file found in the theme folder (you can open it in notepad or a css editor etc) Near the top you will see this line:

.mainpage {
width: 80%;
}

you can change the "80%" to a pixel value ie: "720px" this will give the theme a set width.

Buttons...

The buttons at the top of theme are easily modified, if you look at theme.html in an editor at around line 72 you will see:

<!-- start Buttons -->
<td width="40"><a href="<{$xoops_url}>/"><img src="<{$xoops_imageurl}>but1.png" alt="Home" width="40" height="42" border="0" /></a></td>
<td width="40"><a href="<{$xoops_url}>/modules/mydownloads/"><img src="<{$xoops_imageurl}>but2.png" alt="Downloads" width="40" height="42" border="0" /></a></td>
<td width="40"><a href="<{$xoops_url}>/modules/contact/"><img src="<{$xoops_imageurl}>but3.png" alt="Email" width="40" height="42" border="0" /></a></td>
<td width="40"><a href="<{$xoops_url}>/search.php"><img src="<{$xoops_imageurl}>but4.png" alt="Advanced Search" width="40" height="42" border="0" /></a></td>
<!-- end Buttons -->

You can easily add more buttons by adding new lines. ie: to make a button for your links page, make a new button, name it but5.png and add this line:

<td width="40"><a href="<{$xoops_url}>/modules/mylinks/"><img src="<{$xoops_imageurl}>but5.png" alt="Links" width="40" height="42" border="0" /></a></td>
